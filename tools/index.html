In-App Help


If the game does not work properly here are some solutions (If you need help go to Discord).

  • GDPS_Editor_Bluestacks.cfg

    This will allow you to have platformer controls and practice mode buttons on both game modes.

    GDPS_Editor_Bluestacks.cfg
    Need help?
  • oldApkFix.apk

    If the app closes itself instantly try this apk.

    oldApkFix.apk
  • blackScreen

    (BlueStacks only) If the game stays on a black screen, create a new Nougat-64 instance, full guide here:

    Guide
