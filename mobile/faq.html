1. What is a GDPS?

Geometry Dash Private Server where we have our own levels, our own in-game moderators, working top 100 and so on.

2. Why is this GDPS special?

This GDPS runs on the most recent 2.2 leak, it happened in december 2019 when Robtop leaked new contents of the 2.2 Update on GD LIte.

3. In which platforms can I play the GDPS?

The answer is related to the question above, the leak happened only on the playstore which means this GDPS is only playable on Android and on other platforms where Android can be emulated such as PC using Bluestacks 5 Emulator.

4. What kind of new features does this GDPS offers that are not already in others?

The biggest one is the Platformer mode, as you can see on the sneak peek by robtop himself (https://youtu.be/g4r6BTdU2M0), allows you to create 2D platformer-gameplay. This GDPS has also a lot of new 2.2 Editor features like particle editor, swing copter game-mode, new triggers like song trigger, timewarp trigger etc.
